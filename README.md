# Happy Birthday Fine Boy ❤️

A premium, interactive, mobile-first birthday website built for Odusanya Oluwaferanmi Israel.

## Features
* **Glassmorphism UI:** Elegant frosted glass panels.
* **Animations:** Smooth reveals powered by AOS and GSAP.
* **3D Particles:** Romantic floating particles using Three.js.
* **Interactive Bouquet:** Staggered generation of well-wishes.
* **Responsive:** Fully mobile-first design, perfect for smartphones.

## Setup Instructions
1. Extract all files into a single folder.
2. Create a folder named `assets` inside the main folder.
3. Inside the `assets` folder, add a romantic MP3 file and name it `music.mp3`.
4. Replace the Unsplash image links in `index.html` (under `<section id="gallery">`) with local paths to your personal photos if desired (e.g., `assets/photo1.jpg`).
5. Double-click `index.html` to open it in your browser. No server configuration is required!
